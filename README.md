# wp-reader
Android hybrid app for reading wp blogs

uses ionicframework

# How to use it?

If you want to start using it, you need to have an account in mongo lab

And add you API key in app.js

````
.constant('MONGOLAB_CONFIG', {
	API_KEY: 'YOUR-MONGOLAB-API-KEY',
	DB_NAME: 'wp-reader'
})
````
